# Wall 桌面多视频播放工具（Windows）

Wall 是一款运行在 Windows 桌面端的多视频播放工具，
用于同时播放、展示和管理多个视频或视频流，
适合需要「多画面同时查看」的使用场景。

---

## 🧩 使用场景

- 同时播放多个视频 / 视频流
- 多窗口视频对比
- 桌面监控 / 展示 / 大屏播放
- 不想被浏览器标签页拖慢系统

---

## ✨ 主要特性

- 支持多路视频同时播放
- 桌面级运行，非浏览器环境
- 自由布局 / 窗口调整
- 轻量、稳定、即开即用
